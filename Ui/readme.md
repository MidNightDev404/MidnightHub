# MidnightHub

MidnightHub is a custom hub built using **RedzLib**, authorized by **tlredz**.

## Features
- Custom UI based on RedzLib
- Optimized for performance
- Easy to integrate and use

## Links
- **Discord – RedzLib / RedzHub:** [discord.com/redzhub](https://discord.com/redzhub)  
- **Original UI by tlredz:** [GitHub Repository](https://github.com/tlredz/Library/tree/main/redz-V5-remake)  

## Installation
1. Download or clone this repository.
2. Integrate it with your Roblox project using the RedzLib framework.
3. Launch and enjoy the custom MidnightHub UI.
